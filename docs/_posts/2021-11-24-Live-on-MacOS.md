# 拷贝粘贴路径
这也是一个我在Win/Linux上的常用功能，上传文件时选择文件时非常方便。找到了一个叫`FinderPath`或许能解决问题。在目前最新版本的12.0.1版本系统上勉强上能用。和Win下的资源管理以及Linux下的nautilus一样了，有了地址栏。但是文件选择对话框中并没有，这是一个不完善的功能。
